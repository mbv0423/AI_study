1. Claude codeってなあに？
    - Claude codeとは
        - ひとことで言うと
            - フォルダやファイルの操作からWEBサービスの連携まで、何でもやってくれるAIアシスタント
        - 例
            - PCに常駐する秘書を雇ったようなもの
            - 「この議事録まとめて」「このCSV分析して」と言えば、ファイルを開いて、整理して、保存までやってくれる
            - GoogleWorkspaceなど普段使っているツールともつながる
            - ブラウザを操作して情報を取ってくることもできる
        - できること
            - フォルダの中身をまるごと読んで要約
            - CSVを分析して表やレポートを作成
            - 議事録を整理してTodoを抽出
            - ルールを覚えさせて、毎回同じ品質で出力
            - Slack・Notionなど外部サービスと連携
            - ブラウザを操作して情報収集や入力作業を自動化
        - Manusとのちがい
            - Claude codeは実際に自分のローカルのPC環境にもアクセスできる

1. Claudeの種類
    - Claude = 普通のAIチャット
        - ブラウザやアプリで普段使っているAIチャット
        - ChatGPTやGeminiと同じで、質問→回答という使い方
    - Claude code = これから使ってほしいもの
        - 自分のPCに入っているファイルの操作・外部サービス連携・ブラウザ操作まで「実行」してくれるツール
        - 使い方はいくつかある
            - ブラウザ版:
                - Google Chromeなどのブラウザからアクセスするもの
                - 触りやすそうな見た目だが、GitHubの知識が必要
                - 難易度:中
                
                ![image.png](%E7%AC%AC5%E5%9B%9E%20Claude%20Code%E3%82%92%E5%AE%9F%E9%9A%9B%E3%81%AB%E8%A7%A6%E3%81%A3%E3%81%A6%E3%81%BF%E3%82%88%E3%81%86/image.png)
                
            - デスクトップ版（アプリ版）:
                - アプリとして使えるもの
                - 以下のURLからダウンロードできます
                - 触りやすい見た目なので、特別なこだわりがなければ最初はこれでも良いかも
                - 難易度:低
                
                ![image.png](%E7%AC%AC5%E5%9B%9E%20Claude%20Code%E3%82%92%E5%AE%9F%E9%9A%9B%E3%81%AB%E8%A7%A6%E3%81%A3%E3%81%A6%E3%81%BF%E3%82%88%E3%81%86/image%201.png)
                
            - ターミナル版:
                - できることが一番多い
                - 最初は一番とっつきづらい
                - 難易度:中〜高
                
                ![image.png](%E7%AC%AC5%E5%9B%9E%20Claude%20Code%E3%82%92%E5%AE%9F%E9%9A%9B%E3%81%AB%E8%A7%A6%E3%81%A3%E3%81%A6%E3%81%BF%E3%82%88%E3%81%86/image%202.png)
                
            - Cursor:
                - AIコードエディタのCursorでターミナルを開いて使用
                - Cursor自体にもAIが搭載されているので、Claude codeでうまくいかないときにCursorに任せることもできる
                - 難易度:中
                
                ![image.png](%E7%AC%AC5%E5%9B%9E%20Claude%20Code%E3%82%92%E5%AE%9F%E9%9A%9B%E3%81%AB%E8%A7%A6%E3%81%A3%E3%81%A6%E3%81%BF%E3%82%88%E3%81%86/image%203.png)
                
    - Cowork = 非エンジニア向けの、Claude codeの簡易版
        - Claude Codeを非エンジニア向けに使いやすくしたもの（Desktop版より見た目がわかりやすい）
        - デスクトップ版（アプリ版）からのみ使用可能（ブラウザのClaudeからは使えない）
        - ターミナル不要で手軽だが、カスタマイズ性は低め
        - Claude codeを使えるなら特に必要ない
        - [https://support.claude.com/ja/articles/13345190-cowork%E3%82%92%E5%A7%8B%E3%82%81%E3%82%8B](https://support.claude.com/ja/articles/13345190-cowork%E3%82%92%E5%A7%8B%E3%82%81%E3%82%8B)

1. デモ①:フォルダ整理
    - ファイル名がバラバラで散らかったフォルダをClaude Codeに整理してもらう
        - 使うフォルダ**: `demo1_整理前/`**
            
            中身はこんな状態：
            
            - TODO
            イベント企画書_v2.md
            memo20260208.txt
            告知文draft.md
            議事録.md
            名簿.csv
            slack下書き.txt
            20260215メモ.txt
            mtg_20260201.txt
            アンケート.csv
            - 日付の書き方がバラバラ（`memo20260208` vs `20260215メモ`）
            - 種別が混在（議事録・企画書・名簿・メモが同じ階層にフラットに並んでいる）
            - 英語・日本語・半角・全角が混在
        
        `このフォルダを整理して`と指示してみよう
        
    - 注目ポイント
        - **「整理して」の一言だけで動いた** こと
        - Claude Codeが「自分で判断して」フォルダ構造を決めたこと
        - 結果のレポートが自動で出てきたこと
    - ちいかわで例えると
        
        「ちいかわが『部屋片付けといて』と一言言ったら、うさぎが全部やってくれた。
        
        どこに何をしまったかの報告書まで置いてあった。
        
        ちいかわは帰ってきたときにきれいになった部屋を確認するだけ。」

1. デモ③:さっきの作業をコマンド1つでできるようにしよう
    - 繰り返しやる作業をコマンド一つ打つだけで実行できるようにしてみよう
        - さっきやった分析を`/analyse`とコマンドを打つだけで実行されるようにしてみる
        - どうやるかを決めた.mdファイルをセットで作っておくと精度が上がる（Skillsと言います）
        - コマンドにするのもSkillsにするのもプロンプトで指示するだけ
        - `今後も同じ分析をしたいので、今やった内容をskillsにしてほしいです。またこれを/analyseで呼び出せるようにしてください。`
    - 注目ポイント
        - コマンドを決めておくと毎回長い指示を書く必要がない
        - Skills = 「こういう指示に沿って動いて」と書いたmd.ファイル
            - 一度作れば、毎回同じ品質で同じ作業をやってくれる
            - よくやる作業は全部「コマンド化」できる
            - 不要なコマンドは`このコマンドは削除して`とClaude codeに指示するだけで消せる
